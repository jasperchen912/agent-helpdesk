# Morning Column Template

Use this template for the standing daily WeChat column published at `08:00` China time.

## Best Use Cases

- Daily 8 AM WeChat column
- Stable recurring morning edition
- Habit-forming daily newsletter rhythm

## Structure

1. Fixed title in the format `《X月X日最该看的几件大事》`
2. One short lead that explains the day's main signal
3. `Today's 3 most important stories`
4. `2-3` grouped sections for the rest of the article
5. `What to watch today`
6. Optional image suggestions
7. Source summary

## Article Skeleton

### Title

- Use the exact title pattern `《X月X日最该看的几件大事》` unless the user overrides it

### Lead

- One short paragraph
- Explain the common thread of the day
- Frame why a morning reader should care now

### Today's 3 most important stories

For each item include:

- `Headline`
- `What happened`
- `Why it matters`
- `Source / date`

### Grouped sections

Use `2-3` sections depending on the day. Default section order:

- Tech and business
- World and policy
- Optional watchlist or internet culture with clear impact

For each section:

- Add a short section intro
- Include `2-3` stories
- Keep the same `Headline / What happened / Why it matters / Source / date` structure

### What to watch today

- Add `3-5` concrete watch points for the same day
- Prefer launches, meetings, votes, deadlines, earnings, policy signals, or likely follow-up developments

### Image suggestions

Include only when useful:

- `Cover image suggestion`
- `Section image suggestion`
- `Search prompt`

### Source summary

- List the main outlets or official sources used

## Guardrails

- Write as a morning product, not a generic daily article.
- Prefer the most recent reliable update available by `08:00 Asia/Shanghai`.
- Avoid reusing yesterday's lead story unless there is a material overnight change.
- Keep the rhythm stable enough that readers recognize the column shape every day.
