# Topic Focus Template

Use this template for a single-theme report. This is for prompts such as "only cover AI", "focus on China tech", or "give me today's trade-policy briefing".

## Best Use Cases

- One-sector daily report
- Policy tracking
- Company-cluster updates
- Theme-based newsletter blocks

## Structure

1. One title or up to three title candidates
2. One short thematic lead
3. `1` core narrative section
4. `4-6` selected items grouped under the theme
5. One short `what to watch` close
6. Source summary

## Article Skeleton

### Title

- Make the theme explicit
- Example pattern: `《3月9日AI最该看的几件事》`

### Lead

- Explain why this theme mattered today
- Anchor the story in one clear macro thread

### Core narrative

- Open with one short section that frames the day's biggest shift inside the theme
- Use `2-3` paragraphs maximum

### Selected items

For each item include:

- `Headline`
- `What happened`
- `Why it matters inside this theme`
- `Source / date`

### What to watch

- End with one compact paragraph on the next thing worth watching

### Source summary

- List the main sources used

## Guardrails

- Do not drift into unrelated general headlines.
- Reweight the selection rubric around the requested theme.
- Keep the article coherent as one argument, not a stack of isolated clips.
- If the user asks for depth on one item, expand that item before adding more items.
