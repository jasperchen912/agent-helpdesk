# Tool Boundary

Use this skill for story creation, revision, title work, and editorial QA.

## Stay Here When

- The user wants a new short story for the recurring WeChat series.
- The user wants to rewrite, expand, compress, or review an existing fiction draft.
- The user wants title or frontmatter help for a story.

## Hand Off When

- The user asks to publish or push the finished Markdown into the WeChat draft flow.
  - Hand off to `wechat-publisher`.
- The user wants automation, credential setup, or posting mechanics rather than story writing.
- The request is actually news, commentary, or non-fiction reporting rather than fiction.
