# Prompt Examples

Use these to map short user requests to the right job.

- `写一篇关于最后一班电梯的公众号短篇小说`
  - Job: new standard issue from a direct premise.
- `今晚没灵感，你自己挑一个适合这个系列的题`
  - Job: self-select a premise, then draft a full issue.
- `把这个梗概扩写成能发公众号的短篇`
  - Job: expand a supplied outline into a finished story.
- `这篇太像散文了，改成更像小说`
  - Job: rewrite toward stronger scene, action, and pressure.
- `保留情节，把字数压到 900 字`
  - Job: compress into `flash fiction`.
- `这个故事想写长一点，但先一篇写完`
  - Job: draft an `extended issue`.
- `这个故事想写长一点，拆成三天连载`
  - Job: split the premise into a serial and draft the requested installment.
- `继续写昨天那篇的第二回`
  - Job: serial continuation with continuity preservation.
- `给这篇故事起 3 个标题`
  - Job: title work only.
- `帮我看看这篇能不能发`
  - Job: QA review with findings first.
- `这个系列下一篇别再写办公室了`
  - Job: continuity adjustment before drafting.
