# Title Playbook

Use this file when building or revising titles.

## Default Pattern

Use `《短篇小说｜YYYY.MM.DD｜篇名》` by default.

If the user provides a fixed series name, replace `短篇小说` with that series name and keep the date plus story-title structure.

For serialized stories, keep the same pattern and append the episode marker inside the story-title segment, for example `最后一班电梯（1）`.

## Title Focus Rules

- Keep the story title segment concrete and memorable.
- Prefer `2-10` Chinese characters after the last separator.
- Favor an object, action, place, line of speech, or recurring pressure over an abstract noun.
- Let the title hint at tension without spoiling the turn.

## Good Focus Families

- object: `那把蓝伞`
- action: `最后一班电梯`
- place: `307 号房`
- social trace: `已读之后`
- soft uncanny: `会响的钥匙`

## Avoid by Default

- broad abstractions such as `人生`, `成长`, `命运`, `真相`, `救赎`, `孤独`
- neat moral statements
- mystery bait that says almost nothing
- titles longer than the story can earn

## Serial Markers

- Prefer numeric markers for consistency across longer arcs: `（1）`, `（2）`, `（3）`.
- Use `（上）/（中）/（下）` only when the user explicitly asks for that style.
- Keep the base story title stable across installments unless the user wants each episode renamed.

## Output Rule

- Return one title by default.
- If the user asks for multiple options, keep them to `3-5` and recommend one.
