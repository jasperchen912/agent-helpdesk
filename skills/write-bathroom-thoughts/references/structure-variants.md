# Structure Variants

Use this file when the default four-beat structure would make the series feel repetitive.

## Variant 1: scene-led

Use this as the default.

- `Paragraph 1`: place, object, action
- `Paragraph 2`: concrete observation and pressure shift
- `Paragraph 3`: inward turn
- `Paragraph 4`: quiet landing

Choose this when the trigger is an image, a sound, or a tiny late-evening pause.

## Variant 2: incident-led

Use this when the trigger is one brief social event.

- `Paragraph 1`: the incident itself
- `Paragraph 2`: what made the air change
- `Paragraph 3`: what the incident reveals about distance, fatigue, or restraint
- `Paragraph 4`: quiet landing

Choose this for elevators, queues, convenience stores, transit, office leftovers, or awkward shared spaces.

## Variant 3: thought-led

Use this sparingly when the seed is a small recurring habit rather than a single event.

- `Paragraph 1`: recurring motion or habit in one specific place
- `Paragraph 2`: physical detail that keeps the piece grounded
- `Paragraph 3`: one earned recognition
- `Paragraph 4`: soft landing without doctrine

Choose this only if the first paragraph still contains a place, object, and action. If it opens abstractly, return to the scene-led structure.

## Selection Rule

- Prefer `scene-led` unless there is a clear reason not to.
- Switch structures only to improve freshness or fit the input better.
- Do not blend all three structures in one short piece.
