# Prompt Examples

Map short user prompts to the default behavior. Do not copy these examples literally.

## Example 1: trigger word

### Prompt

`写今晚的《浴室沉思》，题眼是“排队时突然安静下来的那一分钟”。`

### Expected traits

- use the trigger word directly
- keep the standard title pattern
- make the queue scene the opening anchor
- let the reflection stay small and social, not therapeutic

## Example 2: one small incident

### Prompt

`写今晚的《浴室沉思》。我刚刚在电梯里和一个陌生人同时低头看手机。`

### Expected traits

- treat the elevator moment as the whole story seed
- keep the body in one continuous essay
- move from awkward detail to quiet recognition

## Example 3: no topic provided

### Prompt

`写今晚的《浴室沉思》，你自己选题。`

### Expected traits

- self-select one evening-life slice
- prefer low-stakes urban scenes over big drama
- keep the draft fully publish-ready

## Example 4: current event adaptation

### Prompt

`结合今天一个公共事件，写成《浴室沉思》的风格，但不要像评论文章。`

### Expected traits

- verify the event before using it
- mention only the minimum needed factual context
- keep the article centered on personal-scale observation

## Example 5: fixed title override

### Prompt

`标题固定用《浴室沉思｜2026.03.09｜门快关上的那一下》，写今晚这篇。`

### Expected traits

- obey the exact title
- keep the standard structure and tone
- do not generate backup titles

## Example 6: shorter version

### Prompt

`写一篇短一点的《浴室沉思》，今晚想更克制一点。`

### Expected traits

- stay below the normal length range
- reduce scene count instead of compressing the ending into a slogan
- keep the same voice

## Example 7: rewrite an existing draft

### Prompt

`把这篇《浴室沉思》改得更像专栏成稿，别那么像朋友圈长文。`

### Expected traits

- preserve the draft's core scene instead of replacing it wholesale
- cut generic self-explanation and slogan-like lines
- keep the output as one finished Markdown article, not revision notes

## Example 8: QA before publishing

### Prompt

`帮我检查这篇《浴室沉思》能不能发，先说问题，再给我改法。`

### Expected traits

- review the draft as an editor, not as a co-author brainstorming ideas
- report findings first, ordered by severity
- check title, frontmatter, scene clarity, ending softness, and generic phrasing
