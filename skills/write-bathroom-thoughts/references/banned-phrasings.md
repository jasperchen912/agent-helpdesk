# Banned Phrasings

Use this file in the final polish pass. Cut these patterns unless the user explicitly asks for a more运营化的公众号写法。

## Host-Style Openers

- 欢迎来到第X期
- 今天想聊聊
- 你有没有发现
- 你有没有过这样的经历
- 今天我们想聊聊

## Explainer Scaffolding

- 其实很多时候
- 所谓的...本质上就是...
- 承认...并不是...而是...
- 真正的...从来不是...
- 这不是...这是...

## Generic Social Conclusions

- 成年人都不容易
- 在这个什么都快的时代
- 我们也在慢慢失去...
- 别把...当成...
- 最高级的...

## Platform-Worn Concept Hooks

- 报复性熬夜
- 无效休息
- 情绪内耗
- 松弛感
- 情绪价值
- 已读不回也是一种回

## Structural Defaults To Avoid

- `01 / 02 / 03 / 04`
- `写在最后`
- `今日互动`
- `Shower Bubbles`

Treat these as optional add-ons, never as the default body structure.
