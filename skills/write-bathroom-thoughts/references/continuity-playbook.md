# Continuity Playbook

Use this file when choosing a topic or revising a draft against recent issues.

## Check Recent History First

- If the paired OpenClaw memory file is available, read the last `7-14` issue records first.
- If memory is not available, skim the last `7-14` local `《浴室沉思》` drafts before drafting.
- Note four things from those recent issues: scene family, anchor object, opening verb, and ending mode.
- Treat obvious repetition as a quality problem even if the sentences themselves are new.

## Avoid Back-to-Back Repetition

Do not repeat two nights in a row:

- the same primary scene family
- the same anchor object
- the same social pattern
- the same ending move
- the same abstract pressure word such as fatigue, loneliness, or adulthood

## Rotation Matrix

### Scene families

- home interior
- building threshold
- transit and streets
- micro-public interior
- digital residue entering physical space

### Anchor objects

- keys
- wet umbrella
- receipt
- plastic bag
- elevator button
- sink edge
- towel
- bus pole
- freezer door
- half-written message

### Ending modes

- object left behind
- sound stops
- light changes
- body relaxes
- action remains unfinished

## If No History Is Available

- Assume the last issue may already have used a mirror, steam, bathroom, elevator, or queue motif.
- Prefer a non-bathroom scene unless the user explicitly asks for one.
- Prefer an ending mode different from the built-in golden sample.

## Freshness Test

Choose or revise the draft only if at least three of these feel new relative to recent issues:

- new scene family
- new anchor object
- new social distance
- new motion type
- new ending mode
- new sentence cadence
